Available commands:
help                   - просмотр доступных команд

upload *filePath*      - загрузку файла go ftp клиентом на ftp сервер;
::Необходимо ввести название файла. На сервере создастся файл с аналогичным названием
download *filePath*    - скачивание файла go ftp клиентом с ftp сервера;
::Необходимо ввести название файла. На локальной машине создастся файл с аналогичным названием
read *filePath*        - вывод содержимого файла в консоль
mkdir *name*           - создание директории go ftp клиентом на ftp сервере;
rm *name*              - удаление go ftp клиентом  файла на ftp сервере;
ls                     - получение содержимого текущей директории на ftp сервере с помощью go ftp клиента.
